# Linux Befehle

## Grundlegende Befehle

- ls → Dateien anzeigen
- cd → Verzeichnis wechseln
- pwd → aktuelles Verzeichnis anzeigen
- mkdir → Ordner erstellen
- rm → Dateien löschen
- cp → Dateien kopieren
- mv → Dateien verschieben/umbenennen

## Hilfe im Terminal

- man <befehl> → Handbuchseite
- --help → kurze Hilfeanzeige
