# Linux Grundbefehle 🐧

Zurück zum Wiki: README.md

## Navigation
- ls → Dateien anzeigen
- cd → Verzeichnis wechseln
- pwd → aktuelles Verzeichnis

## Dateien & Ordner
- mkdir → Ordner erstellen
- touch → Datei erstellen
- rm → löschen
- cp → kopieren
- mv → verschieben

## Dateiinhalt
- cat → anzeigen
- less → seitenweise
- head → Anfang
- tail → Ende

## Hilfe
- man → Handbuch
- --help → Hilfe
