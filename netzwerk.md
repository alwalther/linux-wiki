# Linux Netzwerk 🌐

Zurück zum Wiki: README.md

## IP & Interfaces
- ip a → IP anzeigen
- ip link → Netzwerkgeräte
- hostname -I → schnelle IP

## Verbindung testen
- ping → Verbindung testen
- curl → HTTP Anfrage
- wget → Download

## Ports
- ss -tulpen → offene Ports
- netstat → alt

## DNS & Routing
- resolv.conf → DNS
- ip route → Routing Tabelle
