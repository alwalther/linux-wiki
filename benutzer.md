# Linux Benutzer & Rechte 👤

Zurück zum Wiki: README.md

## Benutzer
- whoami → aktueller User
- id → User Info
- adduser → erstellen
- deluser → löschen

## Gruppen
- groups → anzeigen
- usermod -aG → hinzufügen

## Rechte
- ls -l → anzeigen
- chmod → Rechte ändern
- chown → Besitzer ändern

## sudo
- sudo → Admin Rechte
